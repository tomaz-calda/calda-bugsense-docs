---
title: "Create reply"
api: "POST /api/v1/projects/{projectId}/issues/{issueId}/comments/{commentId}/replies"
description: "Creates a reply to an existing comment."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="issueId" type="string" required>
  UUID of the issue.
</ParamField>

<ParamField path="commentId" type="string" required>
  UUID of the parent comment to reply to.
</ParamField>

## Request body

<ParamField body="body" type="string" required>
  Reply text (1–10,000 characters).
</ParamField>

<RequestExample>
```json
{
  "body": "Thanks for looking into it! Let me know if you need more details."
}
```
</RequestExample>

## Response

Returns the created reply comment. The `parentCommentId` field will reference the parent comment.
