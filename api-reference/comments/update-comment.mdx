---
title: "Update comment"
api: "PUT /api/v1/projects/{projectId}/issues/{issueId}/comments/{commentId}"
description: "Updates a comment's text. Also updates the comment in Linear."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="issueId" type="string" required>
  UUID of the issue.
</ParamField>

<ParamField path="commentId" type="string" required>
  UUID of the comment to update.
</ParamField>

## Request body

<ParamField body="body" type="string" required>
  Updated comment text (1–10,000 characters).
</ParamField>

<RequestExample>
```json
{
  "body": "Updated: I can reproduce this on Safari 17.2 and 17.3."
}
```
</RequestExample>

## Response

Returns the updated comment. The `edited` field will be `true`.
