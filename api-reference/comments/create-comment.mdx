---
title: "Create comment"
api: "POST /api/v1/projects/{projectId}/issues/{issueId}/comments"
description: "Adds a comment to an issue. The comment is also synced to Linear."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="issueId" type="string" required>
  UUID of the issue.
</ParamField>

## Request body

<ParamField body="body" type="string" required>
  Comment text (1–10,000 characters).
</ParamField>

<RequestExample>
```json
{
  "body": "I can reproduce this on Safari 17.2. Looking into it now."
}
```
</RequestExample>

Returns the created comment object.
