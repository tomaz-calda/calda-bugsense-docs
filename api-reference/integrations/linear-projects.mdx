---
title: "List Linear projects"
api: "GET /api/v1/integrations/linear/projects"
description: "Returns projects from the connected Linear workspace. Admin only."
---

## Authentication

<Icon icon="shield" /> Requires **admin** role

## Query parameters

<ParamField query="teamLeadUserId" type="string" optional>
  Filter projects by a specific team lead's Linear connection.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of Linear project objects.

  <Expandable title="project object">
    <ResponseField name="id" type="string">Linear project ID</ResponseField>
    <ResponseField name="name" type="string">Project name</ResponseField>
    <ResponseField name="url" type="string">Direct link to the project in Linear</ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>
```json 200
{
  "data": [
    {
      "id": "proj_abc123",
      "name": "Client App Redesign",
      "url": "https://linear.app/team/project/client-app-redesign"
    }
  ]
}
```
</ResponseExample>
