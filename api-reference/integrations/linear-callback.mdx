## Authentication

This endpoint does not require authentication. It is called by Linear's OAuth redirect.

## Query parameters

<ParamField query="code" type="string" required>
  Authorization code from Linear.
</ParamField>

<ParamField query="state" type="string" required>
  State parameter for CSRF protection. Must match the stored OAuth state.
</ParamField>

## Behavior

1. Validates the `state` parameter against stored OAuth states (states expire after 10 minutes)
2. Exchanges the `code` for an access token and refresh token via Linear's token endpoint
3. Stores both tokens in the `linear_connections` table (upserts if a connection already exists)
4. Registers webhooks on the connected Linear team
5. Redirects the user to `{FRONTEND_URL}/app/linear/connected` via `302` redirect

The access token expires after 24 hours. BugSense automatically refreshes it using the stored refresh token whenever it is needed.

## Error handling

- Invalid or expired `state` returns `400`
- Token exchange failure returns `500`
