---
title: "List all projects"
api: "GET /api/v1/projects"
description: "Returns all projects with their clients. Admin only."
---

## Authentication

<Icon icon="shield" /> Requires **admin** role

## Response

<ResponseField name="data" type="array">
  Array of project objects with clients. Each project has the same shape as the response from [Create project](/api-reference/projects/create-project).
</ResponseField>
