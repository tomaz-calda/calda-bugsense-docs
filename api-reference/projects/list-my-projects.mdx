---
title: "List my projects"
api: "GET /api/v1/projects/my"
description: "Returns projects the authenticated user has access to."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Behavior

- **Admin users** see projects where they are the team lead.
- **Client users** see the project they are assigned to.

## Response

<ResponseField name="data" type="array">
  Array of project objects. Same shape as [List all projects](/api-reference/projects/list-projects), filtered by the user's access.
</ResponseField>
