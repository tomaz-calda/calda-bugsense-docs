---
title: "Approve report"
api: "POST /api/v1/projects/{projectId}/reports/{reportId}/approve"
description: "Approves a draft report, making it visible to clients and triggering email notifications."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="reportId" type="string" required>
  UUID of the report to approve.
</ParamField>

## Behavior

Approving a report:

1. Changes the report status from `draft` to `approved`
2. Sends email notifications to project clients with the report summary
3. Sends Slack DMs to relevant team members

## Response

Returns the approved report with `status: "approved"`.
