---
title: "Get report"
api: "GET /api/v1/projects/{projectId}/reports/{reportId}"
description: "Returns a single weekly report."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="reportId" type="string" required>
  UUID of the report.
</ParamField>

## Response

Returns a single report object. See [List reports](/api-reference/reports/list-reports) for the full schema.
