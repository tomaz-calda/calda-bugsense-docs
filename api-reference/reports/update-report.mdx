---
title: "Update report"
api: "PUT /api/v1/projects/{projectId}/reports/{reportId}"
description: "Updates a weekly report's narrative content before approval."
---

## Authentication

<Icon icon="lock" /> Requires `Authorization: Bearer <token>`

## Path parameters

<ParamField path="projectId" type="string" required>
  UUID of the project.
</ParamField>

<ParamField path="reportId" type="string" required>
  UUID of the report to update.
</ParamField>

## Request body

All fields are optional, but at least one must be provided.

<ParamField body="headline" type="string" optional>
  Updated headline.
</ParamField>

<ParamField body="summary" type="string" optional>
  Updated narrative summary.
</ParamField>

<ParamField body="highlights" type="string[]" optional>
  Updated highlights (at least one item if provided).
</ParamField>

<ParamField body="risks" type="string[]" optional>
  Updated risks (at least one item if provided).
</ParamField>

<ParamField body="nextSteps" type="string[]" optional>
  Updated next steps (at least one item if provided).
</ParamField>

<RequestExample>
```json
{
  "headline": "Solid week with key bugs resolved",
  "highlights": [
    "Resolved the Safari login bug affecting 30% of users",
    "Reduced average response time to under 24 hours"
  ]
}
```
</RequestExample>
