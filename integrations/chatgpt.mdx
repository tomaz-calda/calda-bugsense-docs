## Overview

BugSense exposes a [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server that lets ChatGPT query your projects, issues, and project health directly through natural language. Users authenticate with their BugSense credentials via an OAuth flow, and ChatGPT can then call BugSense tools on their behalf.

Access is scoped to the user's role — admins see all projects, clients only see their assigned project.

## Available tools

ChatGPT can call these tools through the MCP protocol.

### `list_projects`

List all BugSense projects with their name, team lead, and active issue count. Use this first to discover available projects before querying issues.

## MCP prompts

The server also exposes guided prompts that walk the user through multi-step flows.

### `report_bug`

Guided flow to report a new bug — walks the user through providing project, title, description, platform, and environment step by step.

### `request_feature`

Guided flow to request a new feature — collects project, title, and description step by step.

<Note>
Both prompts accept an optional `projectId` argument. If omitted, the user is asked to pick a project first.
</Note>
